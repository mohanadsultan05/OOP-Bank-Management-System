# 🏦 Bank Management System (C++)

A complete Bank Management System developed in **C++** using **Object-Oriented Programming (OOP)** principles. The project provides a console-based banking application with user authentication, client management, money transfers, transaction logs, and currency exchange.

## ✨ Features

- 👤 User Login System
- 🔐 User Permissions Management
- 📝 Login Register (Activity Log)
- 👥 Client Management
  - Add Client
  - Update Client
  - Delete Client
  - Find Client
  - List Clients
- 💰 Deposit Money
- 💸 Withdraw Money
- 🔄 Money Transfer Between Accounts
- 📄 Transfer Log
- 💱 Currency Exchange
- 💾 File-Based Data Storage

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Standard Template Library (STL)

## 📂 Project Structure

- Client Management
- User Management
- Transaction Module
- Currency Exchange Module
- Login Register
- Transfer Log
- Screen Management
- File Storage

## 🚀 How to Run

1. Clone the repository.
2. Open the project in **Visual Studio**.
3. Build and run the project.
4. Make sure the required text files exist (or create empty ones if needed):
   - Clients.txt
   - Users.txt
   - TransfersLog.txt
   - LoginRegister.txt
   - Currencies.txt

## 📖 OOP Concepts Used

- Encapsulation
- Inheritance
- Abstraction
- Static Members
- Composition
